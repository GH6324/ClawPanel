# ClawPanel Pro v5.2.11

发布时间：2026-03-13

## 前端性能优化

- 路由页面懒加载（Suspense），显著提升首屏加载速度
- legacy agents.bindings 提升到顶层，减少嵌套解析开销

## Agent 与通道稳定性修复

- 修复合并后 agent/channel 流程中的多项稳定性问题
- 支持外部 agentDir 及 `agent` 子目录规范化
- 飞书 requireMention 校验修正为 boolean 类型
- Binding 展开索引改为 nullable 并增加越界保护

## CI 与构建链改进

- 修复 macOS/Windows CI 中 symlink 导致的 agent workspace 测试失败
- 安装脚本与文档统一切换到 Gitee 入口，减少国内用户访问 GitHub 不稳定的影响
- 新增本地构建包安装辅助脚本（Pro/Lite 双版本）
